# Natural Prompt Pack

This file is a natural-language prompt pack for the current system.

The goal is to test what the app already supports without over-biasing prompts toward a single opening style like "Create..." or toward only one family like newspapers or maps.

## Current Template Coverage

The system currently includes support across:

- Newspapers
- Maps
- Bar, pie, and line charts
- Pricing and comparison layouts
- Before/after and split-screen layouts
- Process steps and timelines
- Event promos and testimonial walls
- Text-driven motion scenes
- Cards, lists, quotes, and KPI callouts
- Loading, stream, and news utility screens
- Showcase-style motion layouts

---

## Newspapers
